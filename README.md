# Avance Salon & Spa - Booking System

## Installation
1. `cd server && npm install`
2. Create your `.env` based on `.env.example`.
3. Run `node seed_admin.js` to create your admin account.

## Run Development
`npm run dev`

## Deployment
- **Backend:** Host on Render or Railway.
- **Database:** MongoDB Atlas.
- **Environment:** Ensure all `.env` variables are added to your hosting dashboard.
